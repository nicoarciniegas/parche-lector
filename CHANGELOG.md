# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Reading statistics feature with charts and analytics.
- Review interaction system (likes and comments).
- User profile editing functionality.
- Social feed to see friends' activities.
- Docker Compose configuration for easy deployment.
- Detailed documentation for backend API and database schema.

### Changed
- Improved UI responsiveness for mobile devices.
- Updated authentication flow to use JWT with secure cookies.
- Refactored backend folder structure for better scalability.

### Fixed
- Fixed issue with book search pagination.
- Resolved styling conflicts in the navigation bar.
- Fixed database connection timeouts in Docker environment.
